# KurosioAuctionSystem
Minecraftサーバー内でオークションができるプラグイン


## 使い方 / コマンド集
出品したいアイテムを持ち、  
 ``/kac start <開始価格> [最低入札単位] [半径] ``  

でオークションをスタート。  
指定する半径以内のプレイヤーにオークションの開始とID等が表示されます。  
<デフォルト>  
最低入札単位：1000円  
半径：15ｍ（最大30ｍ）  

``/kac join <ID>``  ー指定IDのオークションに参加  

``/kac bid``　ーオークション参加後、最低入札単位分 入札  

``/kac bid <金額>``　ー指定金額分入札  

``/kac autobid <金額>``　ー指定金額分まで自動入札  

``/kac leave``　ーオークションから離脱 （最高入札者の場合落札者となります。） 

``/kac exlist``　ー出品中のアイテムの情報を表示 (出品者のみ)  

``/kac cancel``  ーオークションを中止 (出品者のみ)    

``/kac help``　　ーコマンド集を表示  

## プラグイン詳細
オークション開始後、出品者のアイテムはサーバーのメモリ上へ預かる形となります。  
``kac-auctions.yml``にも保存されるため二重管理となっています。  
また、アイテムを落札者に送信または出品者に返却される際、インベントリが満杯の場合、該当ユーザー付近の床に落とされるようになっています。


### kac-auctions.yml
オークション出品中の情報は ``kac-auctions.yml`` に保存されます。  
サーバーがオークション実行中にダウンした場合、オークションは中止され、   
復旧後、出品者がサーバーにログイン次第アイテムは返却されます。 
### history.yml
オークションの結果が保存されます。
