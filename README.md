# KurosioAuctionSystem
Minecraftサーバー内等でオークションができるプラグイン


## 使い方 / コマンド集
出品したいアイテムを持ち、  
 ``/kac start <開始価格> [最低入札単位] [半径] ``  

でオークションをスタート。  
指定する半径以内のプレイヤーにオークションの開始とID等が表示されます。  
<デフォルト>  
最低入札単位：1000円  
半径：15ｍ（最大30ｍ）  

``/kac join <ID>``  ー指定IDのオークションに参加  

``/kac bid``　ーオークション参加後、最低入札単位分 入札できる。  

``/kac bid <金額>``　ー指定金額分入札  

``/kac autobid <金額>``　ー指定金額分まで自動入札  

``/kac exlist``　ー 出品中のアイテムの情報を表示 　

``/kac leave``　ーオークションから離脱
